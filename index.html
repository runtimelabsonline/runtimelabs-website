CAP-03
Version Control Support

Packages are distributed via the Unity Package Manager with scoped registry support. All assets use text-based serialization and are fully diffable in Git, Perforce, or Plastic SCM.

UPM scoped registry Git-diffable Perforce compatible
CAP-04
Semantic Versioning

All packages follow SemVer 2.0.0 strictly. Breaking changes only ship in major versions. Minor versions are additive. Patch releases contain only bug fixes — no API surface changes.
